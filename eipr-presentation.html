Hands-Free Dictation Assistant

Executive Summary

This project proposes a hands-free dictation web application that transforms the way students, researchers, writers, and visually impaired users interact with digital content. Instead of reading from a screen, users can upload or paste text which is read aloud in well-paced, repeatable chunks. The app is designed to enhance productivity, accessibility, and focus by supporting customizable text-to-speech playback, intelligent pauses, and user-friendly interaction features.

Product Description

Key Features

  • Upload PDFs or paste custom text
  • Hear content spoken aloud using TTS
  • Experience automatic segmentation of text (~25–28 characters per chunk)
  • Hear each chunk repeated twice with controllable pauses
  • Control speech rate, voice gender/accent, and interval timing
  • Update profile photo and username via a user-friendly settings interface
  • Save playback preferences with a single "Apply" button

Technologies Used

Frontend: React + Tailwind CSS

Backend: Supabase (Authentication, Database, Storage)

Speech Engine: Web Speech API

Deployment: GitHub Pages / Vercel

Mobile APK: Built using Android WebView wrapper
